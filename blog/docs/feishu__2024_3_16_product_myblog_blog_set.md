# 工程简介：

框架使用vitepress ：https://vitepress.dev/

增加了以下功能：

umami数据统计：https://umami.is/

artalk用户评论：https://artalk.js.org/

# 如何启动

在检出工程后：

```ts
git clone https://github.com/ftyszyx/feishu-vitepress.git
```

 进入blog目录,执行下面命令即可

```ts
npm install
npm run dev
```
